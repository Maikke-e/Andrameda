# 🤖 Андромеда v3.0 — Голосовой ассистент на Python

Голосовой ассистент с Telegram-интерфейсом, управлением браузером Яндекс, окнами Windows, напоминаниями и эмоциональным интеллектом.

## ✨ Возможности

### Telegram Бот (aiogram 3)
- **Голосовые команды** — отправляй голосовые сообщения, Андромеда распознает и выполнит
- **Текстовые команды** — классический способ взаимодействия
- **White List** — доступ только для авторизованных пользователей
- **Уведомления** — получай напоминания прямо в Telegram

### 🌐 Управление Браузером Яндекс
- Запуск или переключение фокуса на уже запущенный браузер
- Открытие/закрытие вкладок по названию
- Переключение между вкладками
- Чтение заголовков новостей
- Сохранение статей в закладки
- Подключение через CDP (Chrome DevTools Protocol)

### 🪟 Расширенное Управление Окнами
- Переключение между окнами по названию
- Сворачивание/разворачивание окон
- Расположение окон (слева/справа/на весь экран)
- Автоматическое расположение нескольких окон
- Каскадное размещение

### 🧠 Контекстные Диалоги
- **Расширенная память** — хранит до 10 последних команд (настраивается)
- **Мультимодальный контекст** — учитывает время суток, день недели, предыдущие действия
- **Контекстно-зависимые команды** — одна фраза = разные действия в разных ситуациях
- **Частота команд** — адаптация под часто используемые команды

### 😊 Эмоциональный Интеллект
- **Определение настроения** — по голосу и тексту
- **Адаптивные ответы** — весёлые когда пользователь весёлый
- **Адаптация характера** — Андромеда со временем становится похожа на пользователя
- **Эмоциональная память** — не повторяет шутки, которые не зашли

### 🔊 TTS (Text-to-Speech)
- **Silero TTS** — качественный русский голос
- **Кэширование фраз** — частые фразы генерируются один раз
- **Streaming** — начало ответа до окончания обработки
- **Эмоциональная окраска** — голос меняется в зависимости от настроения

### ⏰ Напоминания, Таймеры, Будильники
- **Напоминания** — "напомни через 5 минут позвонить маме"
- **Таймеры** — обратный отсчёт с уведомлением
- **Будильники** — ежедневные или разовые
- **Повторяющиеся** — ежедневные/еженедельные напоминания

### ⏱️ Анализ Времени Ответа
- Отслеживание каждого этапа (STT, LLM, TTS)
- Статистика по компонентам
- Рекомендации по оптимизации
- Оценка времени для разных типов команд

## 🛠️ Установка

### 1. Клонирование репозитория
```bash
git clone <url-репозитория>
cd andromeda-v2
```

### 2. Создание виртуального окружения
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка окружения
Создай файл `dev.env` на основе примера:

```bash
cp dev.env.example dev.env
```

Отредактируй `dev.env`:
```env
# Telegram Bot
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_WHITE_LIST=your_username,your_user_id

# API Keys
PICOVOICE_TOKEN=your_picovoice_token
GROQ_API_KEY=your_groq_api_key

# Настройки
CONTEXT_MAX_COMMANDS=10
TTS_CACHE_ENABLED=true
```

### 5. Получение API ключей

#### Telegram Bot Token
1. Напиши [@BotFather](https://t.me/botfather)
2. Создай нового бота: `/newbot`
3. Скопируй токен

#### Picovoice Token
1. Зарегистрируйся на [picovoice.ai](https://picovoice.ai)
2. Перейди в Console → AccessKey
3. Скопируй ключ

#### Groq API Key
1. Зарегистрируйся на [groq.com](https://groq.com)
2. Перейди в API Keys
3. Создай новый ключ

### 6. Установка Vosk модели
```bash
# Скачай модель
wget https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip

# Распакуй
unzip vosk-model-small-ru-0.22.zip
mv vosk-model-small-ru-0.22 model_small
```

### 7. Запуск
```bash
python main.py
```

## 📋 Команды

### Браузер
| Команда | Описание |
|---------|----------|
| `открой яндекс` / `запусти яндекс браузер` | Запустить Яндекс Браузер или переключить фокус на уже запущенное окно |
| `закрой вкладку с ютуб` | Закрыть вкладку с YouTube |

|---------|----------|

| `справа телеграм` | Расположить Telegram справа |

| Команда | Описание |
|---------|----------|
| `напомни через 5 минут позвонить` | Напоминание через 5 минут |

## ⚙️ Конфигурация

### Переменные окружения (`dev.env`)
```env
# Telegram
TELEGRAM_BOT_TOKEN=your_token
TELEGRAM_WHITE_LIST=user1,user2,user_id1

# API
PICOVOICE_TOKEN=your_token
GROQ_API_KEY=your_key

# Контекст
CONTEXT_MAX_COMMANDS=10      # Команд в памяти диалога
CONTEXT_TTL=30               # Минут жизни контекста

# TTS
TTS_CACHE_ENABLED=true
TTS_STREAMING_ENABLED=true

# Эмоции
EMOTION_DETECTION_ENABLED=true

# Браузер
BROWSER_TYPE=yandex
BROWSER_DEBUG_PORT=9222
YANDEX_BROWSER_LNK_PATH=C:\Andromeda\bin\Яндекс Браузер с Алисой AI.lnk
```

## 📊 Анализ Времени Ответа

### Ожидаемое время для разных команд:

| Тип команды | Время | Компоненты |
|-------------|-------|------------|
| Простая (статус) | ~0.7с | STT (0.5) + Обработка (0.2) |
| Браузер | ~1.5с | STT (0.5) + Браузер (1.0) |
| LLM разговор | ~2.5с | STT (0.5) + LLM (1.5) + TTS (0.5) |
| Полный pipeline | ~3.0с | Все этапы + накладные расходы |

### Оптимизация:
- **STT**: Используйте vosk-model-small (быстрее)
- **LLM**: Groq имеет низкую задержку (~500мс)
- **TTS**: Кэширование ускоряет частые фразы в 10x
- **Streaming**: Начало воспроизведения до полной генерации

```
│   ├── command_router.py   # Маршрутизация команд

│   └── yandex_controller.py

│   └── window_manager.py
├── tts/                     # Text-to-Speech (Silero)
│   └── tts_engine.py
├── emotions/                # Эмоциональный интеллект
│   └── emotion_engine.py
├── reminders/               # Напоминания, таймеры, будильники

├── cache/                   # Кэш TTS и прочие данные

├── docs/                    # Документация
├── bin/                     # Ярлыки и вспомогательные файлы
├── config.py               # Конфигурация
├── main.py                 # Точка входа
├── requirements.txt        # Зависимости
└── dev.env                 # Переменные окружения
```
