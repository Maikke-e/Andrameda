# Дизайн интерфейса настольного приложения "Андромеда"

## 1. Целевая аудитория

### Основные пользователи
- **Личный пользователь проекта** — создатель и основной пользователь системы

### Потребности аудитории
- Быстрый доступ к функциям голосового ассистента
- Визуальная обратная связь о выполнении команд
- Управление системными функциями через интерфейс
- Минималистичный дизайн без лишних элементов
- Поддержка русского языка

---

## 2. Основные функции приложения

### Функциональные модули
1. **Голосовое управление**
   - Запуск/остановка прослушивания
   - Индикатор распознавания речи
   - История команд

2. **Управление окнами**
   - Список активных окон
   - Быстрое переключение фокуса
   - Свертывание/разворачивание/закрытие
   - Snap-размещение окон

3. **Браузер Яндекс**
   - Список открытых вкладок
   - Переключение между вкладками
   - Закрытие вкладок
   - Открытие URL

4. **Напоминания и таймеры**
   - Создание напоминаний
   - Список активных напоминаний
   - Удаление напоминаний

5. **Эмоциональный интеллект**
   - Текущее эмоциональное состояние
   - История эмоций

6. **Система**
   - Статус компонентов
   - Логи работы
   - Настройки

---

## 3. Выбор технологии реализации

### Рекомендация: **WPF (Windows Presentation Foundation)**

#### Обоснование выбора:

| Критерий | WPF | WinForms | Electron |
|----------|-----|----------|----------|
| Интеграция с Python | Через PyWinForms | Через Python.NET | Через веб-интерфейс |
| Производительность | Высокая | Средняя | Низкая (Chromium) |
| Размер приложения | Маленький | Маленький | Большой (100+ МБ) |
| Современный UI | Да | Ограниченно | Да |
| Русская локализация | Отлично | Отлично | Требует настройки |
| Сложность разработки | Средняя | Низкая | Низкая |

#### Альтернативный вариант: **Electron + React**
- Если нужен кроссплатформенный интерфейс
- Можно использовать существующий FastAPI backend
- Больше ресурсов, но гибкость в дизайне

### Компоненты окна

#### 4.1 Заголовок окна
- Логотип приложения (иконка "Андромеда")
- Название: "Андромеда v3.0"
- Кнопки управления окном (свернуть, развернуть, закрыть)
- Индикатор статуса (зеленый/желтый/красный)

#### 4.2 Меню
- **Файл**: Запуск/остановка, Выход
- **Окна**: Обновить список, Snap окон
- **Браузер**: Обновить вкладки, Открыть URL
- **Напоминания**: Добавить, Показать все
- **Система**: Статус, Логи, Настройки

#### 4.3 Панель навигации (левая)
- Иконки-кнопки для переключения между разделами
- Ширина: 60px (свернуто) / 200px (развернуто)
- Возможность сворачивания в иконки

#### 4.4 Панель инструментов (верхняя)
- Контекстные кнопки для текущего раздела
- Кнопка голосового ввода (центральная, крупная)

#### 4.5 Рабочая область (центральная)
- Динамический контент в зависимости от выбранного раздела
- Карточки с информацией
- Таблицы со списками

#### 4.6 Статусная строка (нижняя)
- Текущий статус системы
- Эмоциональное состояние ассистента
- Время работы
- Индикатор подключения к API

## 6. Интерактивные элементы

### 6.1 Кнопки

#### Типы кнопок:

| Тип | Внешний вид | Использование |
|-----|-------------|---------------|
| Primary | Синяя заливка, белый текст | Главные действия |
| Secondary | Серая рамка, синий текст | Второстепенные действия |
| Icon | Только иконка | Панель навигации |
| Floating Action | Круглая, большая | Голосовой ввод |

#### Состояния кнопок:
- Normal: обычное состояние
- Hover: подсветка, изменение курсора
- Pressed: нажатие
- Disabled: серая, неактивная
- Loading: индикатор загрузки

#### Текстовые поля:

#### Выпадающие списки:
- Список окон

#### Переключатели (Toggle):
- Включение/выключение функций

#### Модальные диалоги:

  - Кнопки: Отмена, Создать

- **Настройки**
  - Вкладки: Общие, Голос, Эмоции, API

  - Иконка вопроса

## 7. Адаптивность под разные разрешения

### Брейкпоинты:

| Размер | Ширина | Описание |
|--------|--------|----------|
| Small | 800px | Минимальная ширина |
| Medium | 1024px | Стандартный ноутбук |
| Large | 1366px | Десктоп |
| XL | 1920px+ | Широкоформатные |

### Адаптивные изменения:

#### При ширине < 1024px:
- Панель навигации сворачивается в иконки
- Шрифты уменьшаются до 13px
- Элементы сетки перестраиваются

#### При ширине < 800px:
- Режим компактного отображения
- Скрываются второстепенные колонки
- Увеличивается отступ между элементами

### Масштабирование:
- Поддержка DPI 100%, 125%, 150%, 200%
- Векторные иконки (SVG)
- Относительные единицы измерения

## 8. Пользовательские сценарии взаимодействия

### Сценарий 1: Управление окнами

```
1. Пользователь нажимает на иконку "Окна" в навигации
2. Система отображает список активных окон
3. Пользователь видит:
   - Название окна
   - Иконку приложения
   - Статус (видимо/свернуто)
4. Пользователь дважды кликает на окно "Telegram"
5. Система переводит фокус на окно
6. В статусной строке появляется сообщение: "Фокус на окно: Telegram"
```

### Сценарий 2: Голосовое управление

```
1. Пользователь нажимает на кнопку микрофона
2. Кнопка меняет цвет на красный, появляется анимация "слушаю"
3. Пользователь говорит: "Открой YouTube"
4. Система распознает речь и отправляет команду
5. В интерфейсе появляется индикатор выполнения
6. Браузер открывает YouTube
7. Кнопка микрофона возвращается в обычное состояние
8. В логе: "Команда выполнена: открыть YouTube"
```

### Сценарий 3: Создание напоминания

```
1. Пользователь нажимает "Напоминания" → "Добавить"
2. Открывается диалоговое окно
3. Пользователь вводит текст: "Позвонить маме"
4. Выбирает время: "через 30 минут"
5. Нажимает "Создать"
6. Напоминание появляется в списке
7. Система подтверждает: "Напоминание создано"
```

### Сценарий 4: Просмотр эмоционального состояния

```
1. Пользователь открывает раздел "Система" → "Эмоции"
2. Видит текущее состояние: "Радость (85%)"
3. График истории эмоций за неделю
4. Возможность очистить историю
```

## 9. Требования к локализации

### Поддерживаемые языки:
- **ru-RU** (основной) — русский
- **en-US** (альтернативный) — английский

### Элементы для локализации:
- Все тексты интерфейса
- Сообщения об ошибках
- Подсказки (tooltips)
- Диалоговые окна
- Форматы дат и времени

### Форматы:
- Дата: `ДД.ММ.ГГГГ` (рус) / `MM/DD/YYYY` (англ)
- Время: `ЧЧ:ММ` (24-часовой формат)
- Числа: разделитель тысяч — пробел (рус) / запятая (англ)

## 10. Требования к доступности (Accessibility)

### 10.1 Клавиатурная навигация
- Tab — переход между элементами
- Enter — активация кнопок
- Space — нажатие переключателей
- Escape — закрытие диалогов

### 10.2 Экранный доступ
- Поддержка NVDA, JAWS
- ARIA-метки для элементов
- Читаемый порядок элементов

### 10.3 Цветовая контрастность
- Контраст текста: минимум 4.5:1
- Альтернативные визуальные индикаторы (не только цвет)

### 10.4 Масштаб текста
- Поддержка увеличения текста до 200%
- Переполнение контента с прокруткой
